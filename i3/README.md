# Overview
This project explores structured information from FEMA’s Disaster Declarations dataset to visualize and analyze patterns in natural disasters across the United States. The goal is to use metrics from an existing information structure (time, location, and disaster type) to generate insights about how natural disasters have changed over time through visualization. This can suggest possible effects of climate change over the years and how it may have influenced natural disasters. 

The dataset is structured as a CSV file and includes fields such as dates, locations, and number of reports, and disaster categories, making it well-suited for aggregation and analysis. Other data fields include FEMA declaration codes and disaster number, which will not be used for this dashboard.
