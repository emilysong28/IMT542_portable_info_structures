# Overview
This project explores structured information from FEMA’s Disaster Declarations dataset to visualize and analyze patterns in natural disasters across the United States. The goal is to use metrics from an existing information structure (time, location, and disaster type) to generate insights about the effects of climate change over time on natural disasters through visualization. 

The dataset is structured as a CSV file and includes fields such as dates, locations, and disaster categories, making it well-suited for aggregation and analysis.

# How to use
The application is an interactive Microsoft Power BI Report. You can open the report via the link provided below, and use the slider to select a year. This will change what is displayed on the map, and help visualize the number of disasters per state within that given year. 


# Possible Ways To Extend 
There are several ways this project could be expanded:
